# Oman Non-Oil Economic Analysis

Reproducible R Markdown analyses of Oman’s non-oil economic sectors.

## Overview
This repository contains selected analyses from my final-year Statistics project at Sultan Qaboos University, focusing on sector-level contributions, stability, and long-term growth of Oman’s non-oil economy under Oman Vision 2040.

## Tools
- R
- R Markdown

## Data
Publicly available data from the National Centre for Statistics and Information (NCSI), Oman.
